# Tujuan dari program adalah untuk menciptakan sistem sederhana untuk penyewaan motor untuk tiga jenis kendaraan: motor matic, motor trail, dan motor sport.  Tarif harian untuk masing-masing jenis motor berbeda: 50000, 100000, dan 75000.  Sesuai kondisi tertentu, program menawarkan variabel asuransi dan diskon yang dapat meningkatkan atau mengurangi harga sewa, kemudian diminta untuk menggunakan perintah input untuk memasukkan kupon. Namun, terdapat kesalahan sintaks pada bagian ini, yaitu nama = int(input("masukkan kupon: ", kalimat)). Jika diperbaiki, algoritma selanjutnya akan melihat jenis motor yang dipilih, mengalikan biaya sewa dengan asuransi, menambahkan atau mengurangi diskon tergantung pada situasi, dan akhirnya menampilkan biaya total penyewaan.
